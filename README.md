1. Скачиваем драйвер CP2102 https://myrobot.ru/downloads/driver-cp2102-esp32.php
2. Разархивируем архив в любую папку
3. Заходим в диспетчер устройст - Win+R, вводим devmgmt.msc
4. Находим CP2102 USB to UART
5. Жмем правой кнопкой мыши, обновить драйвер, поиск драйвера на этом компьютере
6. Выбираем папку, в которую разрахвирировали драйвер
